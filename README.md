# Corevia Website

One-page marketing site for Corevia.

## Stack
- Vite
- Vanilla JS
- CSS

## Local development
```bash
npm install
npm run dev
```

## Production build
```bash
npm run build
```

Build output goes to `dist/`.

## Cloudflare Pages
Create a new Cloudflare Pages project connected to the GitHub repo and use:

- **Framework preset:** Vite
- **Build command:** `npm run build`
- **Build output directory:** `dist`
- **Root directory:** `corevia-site`
